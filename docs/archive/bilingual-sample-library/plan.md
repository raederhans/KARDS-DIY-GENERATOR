# Plan

## Goal

让示例卡库中的 69 张普通卡和 5 张总部卡都具备完整的英文、简体中文卡面内容，并在界面语言切换时展示同语言的示例内容与参考卡图。

## Scope

- 审计公开参考包中的示例 JSON、整卡参考图和 5 张总部示例。
- 建立按 `en` / `zh` 索引的示例卡数据与参考图路径。
- 让未被用户编辑的已加载示例随界面语言切换；用户编辑后不自动覆盖。
- 扩展现有单元测试、公开资源闭包校验和构建验证。

## Sources of truth

- `src/devPreviewCatalog.ts` 的示例目录与语言解析契约。
- `public/reference-pack/v1/` 的可发布示例数据和参考卡图。
- `.runtime/kards-private-assets/sources/craftsoul-data.json` 的上游多语言卡牌字段，仅作为本地生成来源，不进入发布包。
- `tools/verify_dist_private_boundary.mjs` 的公开资源闭包规则。

## Stages

- [x] Stage 1: 审计当前示例数量、语言字段和切换路径。
- [x] Stage 2: 生成并接入普通卡与总部卡的双语内容和参考图。
- [x] Stage 3: 实现语言切换时的安全示例刷新。
- [x] Stage 4: 运行目标测试、完整验证与本地 HTTP 检查。

## Acceptance criteria

- [x] 69 张普通卡在 `en` 与 `zh` 下都有非缺失的标题、正文（允许规则上本来为空）和正确关键词语言。
- [x] 5 张总部卡在 `en` 与 `zh` 下都有标题与正文。
- [x] 每张示例在两种语言下都解析到存在且签名有效的参考卡图。
- [x] 加载一个示例后切换语言，未编辑的卡面和右侧参考图同步切换；用户编辑的卡不被语言切换覆盖。
- [x] 目标测试、TypeScript、生产构建和严格发布资源闭包校验通过。

## Non-goals

- 不改变普通用户导入卡牌、草稿或本地卡库的语言语义。
- 不修改示例卡之外的 UI 翻译。
- 不引入新的运行时依赖或远程热链。

## Closeout

任务于 2026-07-21 完成。完整验证和浏览器双语切换验证均通过，本地 UI 继续由 `http://127.0.0.1:5173/` 提供。
